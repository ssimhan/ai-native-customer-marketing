Layer 01 — Research
Detect
Before I can orchestrate anything, I need to know what is actually happening. Research stops being a quarterly readout — it becomes a live operating input.
Research → Live Intelligence
Detect layer
A lot of lifecycle marketing runs on fuzzy definitions that calcified a long time ago.
Working definition — "Power User"
A person who did 50+ high-intent actions in a 28-day period.
Fuzzy definition → Fuzzy segmentation → Fuzzy messaging → Expansion built on proxies
Detect layer
Before: the old motion
1Remember whether anyone studied this before
2Ask around. Find a stale deck.
3Realize the original question was too broad
4Wait — the data scientist is on vacation
5Weeks later: receive a large Excel file
After: a living intelligence problem
1Surface prior analyses instantly
2Pull up related thinking that already exists inside the company
3Refine the question and accelerate query work
4Keep interrogating the answer — not just first pass
5Treat the question dynamically until the concept itself gets smarter
Detect layer
The value was not "AI helped us analyze faster." The value was that a foundational concept got smarter — and everything downstream improved.
Segmentation became more meaningful
Education strategy tied to actual maturity steps
"Who should we nudge?" decisions became more grounded
Expansion thinking became less anecdotal
Research became a live operating input, not a quarterly readout
Layer 02 — Operations
Decide
Once I have better signal, I need a better decision layer sitting on top of it. This is where I think customer marketing quietly loses the most leverage.
Operations → Structured Decision Support
Decide layer
What you actually receive is not a brief. It is fragments.
CS NoteA certain cohort is not adopting the new workflow feature...
Slackquick question — should we build content for this?
PMmaybe this is actually an onboarding issue?
Supportwe're seeing a pattern in tickets around this too
Salescan we get a quick recommendation before Thursday?
At that point, the real work is not content. The real work is sense-making.
Decide layer
The interpretive questions that someone has to answer before anything useful can happen:
What actually happened? Who is affected?
Is this big or small? Is it a one-off or a pattern?
Is this an education problem, a product problem, a messaging problem, or an instrumentation problem?
What should happen next?

You stop responding to whoever pinged last. You start building an operating layer — the team that helps the business reduce confusion, not just absorb it.

Customer marketing starts to look less like a content function and more like an orchestration layer.
The orchestration layer
These are not just analytics observations.
They are triggers.
Customer adopts a feature Admin stalls Champion shows unusual engagement Region is lagging A use case is taking off New feature lands, intended audience not moving Expansion signal detected
Each trigger drives a different kind of response: education, outreach, proof collection, executive communication, in-product nudges, community moments, expansion motion. Customer marketing becomes the team that connects signal to action.
The compounding effect
In the old model, the team gets credit for heroics. In the new model, you build systems that get smarter over time.
Did we ship something good?
→ Did we build a system that makes the next thing easier?
Did we create enough content?
→ Did we reduce dependency on tribal knowledge?
Did we answer the ask?
→ Did we convert recurring chaos into a standard operating pattern?
Did we run the webinar?
→ Did we make a core definition smarter?
Did we hit the deadline?
→ Did we make it easier for the next person to do good work?
Did we save the account?
→ Did we make a learning asset more reusable?
None of this happens just because you bought an AI tool. You need three things.
01
Instrumentation
If your data and signals are a mess, AI will help you produce polished confusion faster. The output can look sophisticated. But if the underlying inputs are weak, the result is just more articulate confusion.
02
Judgment
You still need humans who know what good is, what a useful brief looks like, and what should never be automated. AI can help you generate options. It cannot own the standard of quality. That is still our job.
03
Systems thinking
If you only use AI as a one-off assistant, you will get one-off gains. If you use it to redesign the flow of work, you get a function that scales exponentially. That is the real shift.
The idea I want to leave you with
The most interesting use of AI in customer marketing is not acceleration. It is redesign.
